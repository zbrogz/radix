Radix Converter

This tool allows conversion between base 10, base 16, and base 2 representation of signed 64-bit integers. Negative values are encoded in two's compelement. For example, -5 in decimal is displayed as ff ff ff ff ff ff ff fb in hex and 11111111 11111111 11111111 11111111 11111111 11111111 11111111 11111011 in binary. The negation symbol - can be placed at the beginning of a value to compute negation. For decimal values, the symbol will simply remain. For hex and binary, the two's complement will be computed and displayed. The two's complement (negative) of a signed binary is computed as follows: Flip the bits and add one. For example, 5 is 000...000101 and -5 is 111...111010 + 1 = 111...111011.
